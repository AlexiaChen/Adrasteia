服务器端程序：
1、后台自动启动的服务，damond方式启动，可通过config文件对其运行参数做配置管理
   
2、后台服务程序数据加载功能及流程：
   1) 定时根据XML配置文件，加载磁盘上的数据，判断当前数据是否新数据
        a) 将指定的某个文件保存到磁盘上
   2) 当前是新数据文件，对其做解析处理：
        a) 解析该文件
        b) 将数据存入到Redis中
        c) 将其转换为Grib2格式，保存到磁盘上

3、后台服务程序Socket侦听功能及处理流程

   2) 接收客户端数据，验证消息长度和消息类型，验证是否为预报数据
